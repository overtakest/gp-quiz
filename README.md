# 📚 ГП Тесты — Telegram Mini App

Тренажёр по **гигиене питания**: 822 вопроса, 5 типов, экзаменационные варианты и
мнемонические подсказки «Tips & Tricks». Светлая/тёмная тема, анимации, оптимизирован
под телефон и ПК.

## Что внутри

| Вкладка | Что делает |
|---|---|
| **Вопросы** | Вся база (822 вопроса) вперемешку, варианты ответов тоже перемешиваются. После ответа — правильный ответ + подсказка для запоминания. Поиск, фильтры по типу/статусу, отметка «выучено». |
| **Экзамен** | Кнопка генерирует вариант: 40 случайных вопросов, таймер 25:00. Разбор с подсказками после сдачи. История попыток. |
| **Профиль** | Данные Telegram-пользователя, кольцо прогресса, статистика: выучено / осталось / точность / экзамены / лучший балл. Выбор оформления (4 дизайна) + светлая/тёмная тема. |
| **🛠 Админ-панель** | Обзор базы, редактор вопросов (добавить/изменить/удалить + подсказки), экспорт/импорт `questions.json` и `tips.json`. |

Типы вопросов: выбор одного/нескольких, верно/неверно, соответствие, число, ответ словом.

**Оформление (4 дизайна, выбор в Профиле, сохраняется на устройстве и синхронизируется через Telegram CloudStorage):** Классик (индиго), Клиника (мятная), Моно (бумага + засечки), Фокус (тил + янтарь). Каждый со светлой и тёмной темой. Файлы — в `assets/skins/`.

## 🚀 Как запустить как бота (GitHub Pages)

### Шаг 1. Выложить приложение на GitHub Pages
1. Создай на GitHub **публичный** репозиторий, например `gp-quiz`.
2. Загрузи в него **содержимое папки `gp-quiz/`** (файлы `index.html`, `assets/`, `data/`, `bot/`).
   Через сайт: *Add file → Upload files* → перетащи всё → *Commit*.
   Или через git:
   ```bash
   cd gp-quiz
   git init && git add . && git commit -m "ГП Тесты"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/gp-quiz.git
   git push -u origin main
   ```
3. В репозитории: **Settings → Pages** → *Source: Deploy from a branch* → ветка **main**, папка **/(root)** → *Save*.
4. Через ~1 минуту сайт будет доступен по адресу:
   **`https://ТВОЙ_ЛОГИН.github.io/gp-quiz/`** — открой его в браузере, чтобы проверить.

### Шаг 2. Создать бота в @BotFather
1. Открой [@BotFather](https://t.me/BotFather) → `/newbot` → задай имя и username → получи **BOT_TOKEN**.

### Шаг 3. Привязать Mini App к боту
**Вариант А — автоматически (скрипт):** нужен Node.js 18+.
```bash
cd gp-quiz/bot
cp .env.example .env          # Windows: copy .env.example .env
# впиши в .env свой BOT_TOKEN и WEBAPP_URL (адрес с шага 1)
node setup-bot.js
```
Скрипт привяжет кнопку меню к Mini App и настроит команды/описание.

**Вариант Б — вручную в BotFather:**
`/mybots` → выбери бота → **Bot Settings → Menu Button → Configure menu button** →
вставь URL `https://ТВОЙ_ЛОГИН.github.io/gp-quiz/` и текст кнопки. Готово.

### Шаг 4. Открыть
Зайди к своему боту в Telegram, нажми кнопку меню — откроется приложение. 🎉

## 🛠 Админ-панель

- **Доступ:** впиши свой числовой Telegram ID в `assets/config.js` (`adminIds: [12345678]`) —
  ID виден на вкладке «Профиль». Тогда админка всегда твоя.
- **Без правки кода:** на вкладке «Профиль» **7 раз нажми на строку версии** внизу — включится админ-режим на этом устройстве.
- Правки базы хранятся **локально** (в этом браузере/Telegram). Чтобы обновить базу **для всех**:
  Админ → **Данные** → *Экспорт* → положи скачанные `questions.json` / `tips.json` в папку `data/`
  репозитория и перезалей на GitHub. (Для общей live-базы без экспорта нужен отдельный бэкенд.)

Ты как админ полностью пользуешься всеми вкладками и тренируешься наравне.

## 🔧 Пересборка базы (для разработчика)

```bash
cd gp-quiz/bot
python prep_data.py      # quiz-db-*.json  ->  data/questions.json
python split_batches.py  # нарезка на батчи для генерации подсказок
python merge_tips.py      # собрать подсказки  ->  data/tips.json
```

## Приватность
Прогресс хранится в **Telegram CloudStorage** (синхронизируется между устройствами) с
запасным вариантом в localStorage. Внешних серверов и сбора данных нет — чистая статика.
